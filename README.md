## 📌 Project Overview
[cite_start]This project was developed during my **Data Engineering Internship at BBI**[cite: 26]. [cite_start]I designed and deployed an automated data ingestion pipeline using **Apache Airflow** and **Python**, transitioning operations from manual CSVs to a centralized **SQLite** database[cite: 27].

## 🛠️ Tech Stack
* [cite_start]**Orchestration:** Apache Airflow [cite: 27]
* [cite_start]**Language:** Python [cite: 27]
* [cite_start]**Database:** SQLite [cite: 27]
* [cite_start]**Data Libraries:** Pandas, Matplotlib [cite: 29]
* [cite_start]**State Management:** JSON-based local storage [cite: 37]

## 🚀 Key Features
* [cite_start]**Automated ETL:** Developed dynamic scripts to automate real-world ETL orchestration[cite: 29].
* [cite_start]**Stateful Tracking:** Built a system to simulate real-world logistics data generation and maintain active shipment states[cite: 37].
* [cite_start]**SCD Type 2 Logic:** Implemented Slowly Changing Dimension logic to automatically transition shipment statuses while preserving historical records[cite: 38].
* [cite_start]**BI Reporting:** Created a workflow to process rolling 10-day shipment data and generate multi-page PDF dashboards[cite: 39].

## 📈 Impact
* [cite_start]Enabled **100% data traceability** for stakeholders through a centralized system[cite: 28].
* [cite_start]Replaced manual, error-prone CSV operations with a fully automated pipeline[cite: 36].
